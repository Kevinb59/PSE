# Moriarty — Présentation PSE

Présentation interactive pour la commission de création d'entreprise (PSE), développée avec **Next.js**, **GSAP**, **Framer Motion** et **Lenis**.

## Fonctionnalités visuelles

- Intro cinématique animée au chargement
- Particules interactives en canvas (réseau connecté)
- Smooth scroll premium (Lenis)
- Révélations au scroll (GSAP ScrollTrigger)
- Titres animés caractère par caractère
- Cartes 3D tilt au survol
- Navigation clavier (flèches, Page Up/Down, Home, End)
- Mode présentation plein écran
- Barre de progression + navigation latérale
- Export PDF via impression navigateur

## Démarrage local

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).

## Déploiement Vercel

1. Pousser le projet sur GitHub
2. Importer le dépôt sur [vercel.com](https://vercel.com)
3. Vercel détecte automatiquement Next.js — aucune config supplémentaire

## Raccourcis

| Touche | Action |
|--------|--------|
| ↓ / Espace | Slide suivante |
| ↑ | Slide précédente |
| Home | Retour au début |
| End | Aller à la conclusion |
